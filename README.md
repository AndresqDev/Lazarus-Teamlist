# Teamlist For Lazarus HCF

## Simple addon for add teamlist to Lazarus
Download .JAR:
- > https://discord.gg/jr4A3y3nzq

Features:
- Configurable
- Optimized (Update rate is configurable)
- Simple & Easy to setup!

Depends:
- ViaVersion

Libs for compile this src:
ViaVersion
Lazarus JAR
Bukkit/Spigot/SpigotApi
